# odin-recipes

This project is a website with recipes

In this project I have demonstrated my skills
of being able to create an html boilerplate
and using all the basic html elements and tags
including heading tags, ordered and unordered list tags,
the image tag and the anchor tag, also the tags used to declare
each html document <!DOCTYPE html> as well as the html element,
the head element and the body element, the title element and the meta element within the head element
